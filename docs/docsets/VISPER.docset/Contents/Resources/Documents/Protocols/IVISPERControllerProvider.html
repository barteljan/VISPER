IVISPERControllerProvider

@protocol IVISPERControllerProvider <NSObject>

An instance that provides a view controller for a specifc route-pattern

  • -controllerForRoute:routingOptions:withParameters:

    Provide provide a view controller for a specifc route-pattern, return nil if you are not responsible for the given route pattern

    Declaration

    Objective-C

    - (UIViewController *_Nullable)
controllerForRoute:(NSString *_Nonnull)routePattern
    routingOptions:(NSObject<IVISPERRoutingOption> *_Nonnull)options
    withParameters:(NSDictionary *_Nonnull)parameters;

    Swift

    func controller(forRoute routePattern: String, routingOptions options: NSObject, withParameters parameters: [AnyHashable : Any]) -> Any!

    Parameters

    routePattern

    The route pattern for which a controller should be provided
    options

    The given routing options to describe the presentation mode in which the controller will be presented
    parameters

    An dictionary of parameters for this route (use them to configure your VC)

    Return Value

    Returns a ViewController if the provider is responsible for this route pattern, or nil otherwise …
